# Executable trust-boundaries session

This record backs the 2026-07-11 plan entry in the program wiki log.

The session separated source-row recovery from public-profile admission for the
42 Cardano and 52 Midnight predicates. It kept semantic artifact slots immutable
until domain-bound authorization supplies exact circuit, verifier-key, SRS, and
transcript values.

It also required two independent implementations to replay every KZG
contribution and verify the selected contribution proof, declared G1 and G2
powers, cross-group consistency, degree, prefix, and final sealed bytes.

For repository control, it selected one-shot authenticated canonical Git seeding,
bounded pack quarantine, opaque credential handles, exact-stream remote
confirmation bundles, and cumulative predecessor imports. OpenSpec closure uses
a relocation-only archive candidate so no receipt binds a tree containing
itself.
