# Public live-pass is the only success state

The program completes only after both selected public testnets confirm real
proof-authorized destination state changes for every proof-template family
authorized in each direction.

A structural harness pass, local proof, fixture flow, project-operated source
root, custom chain fork, or degraded lab deployment can supply engineering
evidence. None can close the program.

If an unmodified public chain lacks a required root, proof relation, verifier
surface, or state transition, the controller records the reproducer and blocks
the dependent program. It does not change the outcome definition.
